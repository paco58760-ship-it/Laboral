🔐 Contraseña de acceso

Esta contraseña protege la app. Cámbiala por una segura.

📦 JSONBin — conexión de datos

1. Ve a jsonbin.io y crea una cuenta gratis.
2. Crea un nuevo Bin con el contenido [].
3. Copia el Bin ID (en la URL) y tu Master Key (en Account → API Keys).

⚠️ Primer uso

Tras configurar JSONBin, entra con tu contraseña y pulsa Publicar para subir los empleos actuales de Labora al bin. A partir de ahí, Labora los cargará automáticamente.
